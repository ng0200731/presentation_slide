Layout Switching

How should users switch between Grid and Linear (1 image per line)?

A

Toggle Buttons in Toolbar

Two icon buttons (grid icon / list icon) in the element toolbar when the image gallery is selected. Click to switch layout instantly.

B

Dropdown in Style Panel

A "Layout" dropdown in the properties panel with options: Grid (2-col), Grid (3-col), Grid (4-col), Linear. More precise control.
