Design: Multi-Image Gallery Element

Approach A — self-contained image element with gallery capabilities

1. Data Model

Element type: "image" (replaces current single-image)

content:

[
  { "url": "/uploads/abc123.jpg", "caption": "" },
  { "url": "/uploads/def456.png", "caption": "" }
]

styles:

{
  "layout": "grid",        // "grid" | "linear"
  "thumbSize": "medium",   // "small" | "medium" | "large"
  "gap": 8,                // spacing between thumbnails (px)
  "columns": 3             // only for grid layout
}

Backward compat: if content is a string (old format), auto-wrap to [{url: content, caption: ""}]

2. Upload & Input Methods

Drag & Drop

Drop zone on the empty gallery or on existing gallery area. Accepts image files. Shows hover highlight.

Ctrl+V Paste

Listen for paste events when gallery is active. Handles clipboard images (from screenshots, copied files). Uses Clipboard API.

Browse Button

Hidden file input triggered by "Add Image" button in toolbar. Accepts image/* types. Multiple file selection enabled.

4. Server: Upload Endpoint

POST /api/upload — multipart/form-data

  • Accepts single or multiple files (field name: images)
  • Validates: image/* mime type, max 10MB per file
  • Saves to server/uploads/ with unique filename
  • Returns: { urls: ["/uploads/abc123.jpg", ...] }

Serves static: /uploads/* mapped to server/uploads/

5. Component Structure

components/
  Elements/
    ImageElement.tsx      ← rewritten: multi-image gallery
  Toolbar/
    ImageToolbar.tsx      ← NEW: grid/linear toggle + size presets + add btn

ImageElement responsibilities:

  • Render grid or linear layout based on styles.layout
  • Empty state: drop zone with "Drop images here, paste, or browse"
  • Image thumbnails with hover overlay (delete, caption edit)
  • Drop zone overlay on dragenter (even with existing images)
  • Paste listener when element is active

ImageToolbar responsibilities:

  • Grid/Linear toggle buttons
  • Small/Medium/Large size selector
  • "Add Image" button → triggers hidden file input
