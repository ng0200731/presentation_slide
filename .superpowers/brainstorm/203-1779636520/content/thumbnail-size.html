Thumbnail Resize: How should users control size?

When you say "thumbnail preview can be resized" — who controls it and how?

A

Global Size Slider

A slider control in the toolbar/panel. Adjusting it changes ALL thumbnails uniformly. Grid adjusts columns automatically based on size.

B

Per-Image Resize

Each image thumbnail can be individually resized (drag handles on corners). Images can have different sizes within the same gallery.

C

Preset Size Options

Choose from preset sizes: Small, Medium, Large. All thumbnails use the same selected size. Simple and predictable.
